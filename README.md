# Project 3 – Retail Sales Analytics Pipeline

## Project Overview

This project analyses retail sales data using Python, Pandas, and SQL to generate useful business insights and a management summary report.

The aim was to transform raw sales records into clear information that supports business decision-making.

## Business Problem

RetailMart UK records sales from different stores, regions, and products.

Management wanted answers to questions such as:

- What is the total revenue?
- What is the average revenue per sale?
- Which store generated the highest revenue?
- Which product generated the highest revenue?
- Which region performed best?
- What was the highest individual sale?
- What was the lowest individual sale?
- Which store recorded the most sales?
